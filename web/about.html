About the method

The method used in this program is called Gauss-Jordan elimination. It is one of the most efficient ways to solve large systems of equations quickly. Let's start with this matrix:

( 3 7 4 3)

(-5 3 9 -1)

( 3 5 1 0)

First, multiply row 1 by 2:

  • R1*2 = (6, 14, 8, 6)

Then, add row 1 and row 2:

  • R1+R2= (1, 17, 17, 5)

The new matrix looks like this:

( 1 7 4 3)

(-5 3 9 -1)

( 3 5 1 0)

Now, the first entry has become a 1. This is the premise of Gauss-Jordan elimination. The end goal is to keep multiplying, dividing, and adding rows together get a matrix that looks like this:

(1 0 0 2.84)

(0 1 0 -2.14)

(0 0 1 2.18)

This could represent a system of equations that looks like this:

( 1a +7b +4c = 3)

(-1a +3b +9c = -1)

( 3a +5b +1c = 0)

And the solution would be:

(a = 2.84)

(b = -2.14)

(c = 2.18)

It is very systematic and easy to predict, and that makes it very powerful.
